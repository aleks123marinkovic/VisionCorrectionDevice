# Vision Corrector

Holds SolidWorks, STL, .3mf, .STEP, and related files for the Head Coil-Mounted Vision Correction Device for Magnetic Resonance Imaging.

First, if you don't have a slicer, download the latest version of OrcaSlicer by following the ReadMe for your system: https://github.com/SoftFever/OrcaSlicer?tab=readme-ov-file 

Download the .3mf file of the vision corrector and open it in your preferred slicer or OrcaSlicer.

Adjust the settings for the layer height to be 0.08 mm or as close to 0.08 mm as possbile. The Bambu X1 Series goes down to 0.08 mm, which we found works very well.

Adjust wall loops to 5 and infill to 30-40%.

Only activate supports for the lens holder tops and the mount.
